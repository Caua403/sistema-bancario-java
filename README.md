# sistema-bancario-java

Este projeto simula um sistema bancário simples em java.

Os conceitos usados neste projeto foram: 
Orientação a objetos;
Encapsulamento;
Herança;
Polimorfismo;
Métodos.

As funcionalidades dele são:
Criação de Conta Bancária;
Métodos de saque e depósito;
Conceitos de Conta Poupança, sendo uma Conta Bancária com rendimento mensal;
Conceitos de Conta Especial, sendo uma Conta Bancária com limite adcional para saque;

Objetivo do projeto: Estudo e prática de Orientação a Objetos em java.
